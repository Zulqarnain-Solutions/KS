# 🚀 KS Academy – Full Stack Website

## 📌 Overview

KS Academy is a full-stack web designed to provide users with access to courses, authentication, and communication features. It is built with modern technologies focusing on performance, scalability, and clean user experience.

---

## ⚙️ Tech Stack

### 🎨 Frontend

* React (Vite)
* Zod (Validation)
* Tailwind CSS (UI Styling)

### 🖥️ Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication
* Cookies-based Authentication & Authorization

---

## ✨ Features

* 🔐 User Authentication (Login & Register & Logout)
* 🛡️ Protected Routes with JWT
* 📚 Courses Display
* 📚 Courses Enrollment
* 📩 Contact Form (Send Messages to Admin)
* ⚡ Fast & Optimized Frontend (Vite)
* ✅ Form Validation using Zod
